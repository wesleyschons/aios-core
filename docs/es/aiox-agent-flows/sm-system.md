## Vision General

**@sm** (River) es el agente Scrum Master del AIOX, responsable por:

- Facilitacion de ceremonias
- Remocion de impedimentos
- Metricas de equipo
- Mejora continua

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*sprint` | Gestionar sprint |
| `*standup` | Facilitar standup |
| `*retro` | Conducir retrospectiva |
| `*metrics` | Ver metricas |
