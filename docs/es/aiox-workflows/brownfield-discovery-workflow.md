## Vision General

Workflow para analizar proyectos existentes, incluyendo:

- Analisis de codebase
- Mapeo de arquitectura
- Identificacion de deuda tecnica
- Documentacion del estado actual

## Cuando Usar

- Proyecto existente desconocido
- Antes de hacer cambios significativos
- Onboarding en proyecto legacy

## Agentes Participantes

| Agente | Rol |
|--------|-----|
| @analyst | Investigacion y analisis |
| @architect | Revision de arquitectura |
| @dev | Analisis de codigo |
| @qa | Evaluacion de calidad |
