## Vision General

Workflow para evolucionar frontends existentes, incluyendo:

- Nuevas paginas/componentes
- Refactoring de UI
- Actualizacion de design system
- Mejoras de UX

## Cuando Usar

- Cambios en frontend existente
- Nuevas features de UI
- Despues de brownfield-discovery
