# Comunidad de Synkra AIOX

> 🇧🇷 [Versão em Português](COMMUNITY-PT.md)

¡Bienvenido a la comunidad de Synkra AIOX!

Estamos construyendo juntos el futuro del desarrollo orquestado por IA.

## Nuestros Valores

- **Colaboración sobre competencia** - Crecemos juntos
- **Inclusión** - Todos son bienvenidos sin importar su nivel de experiencia
- **Transparencia** - Discusiones abiertas, decisiones abiertas
- **Calidad** - Nos importa hacer las cosas bien

## Primeros Pasos

### Primeros Pasos

1. Dale una estrella al repositorio
2. Lee el [README](README.md)
3. Configura tu [entorno de desarrollo](CONTRIBUTING.md#getting-started)
4. Preséntate en [Discussions](https://github.com/SynkraAI/aiox-core/discussions)

### Encuentra tu Primera Contribución

- Busca issues etiquetados como [`good-first-issue`](https://github.com/SynkraAI/aiox-core/labels/good-first-issue)
- Revisa [`help-wanted`](https://github.com/SynkraAI/aiox-core/labels/help-wanted) para tareas más complejas
- Explora las [Discussions abiertas](https://github.com/SynkraAI/aiox-core/discussions) para ayudar a otros

## Canales de Comunicación

### GitHub Discussions (Principal)

Nuestro centro de comunicación principal para todos los repositorios AIOX:

- **Announcements** - Actualizaciones del proyecto de los mantenedores
- **General** - Discusiones generales sobre AIOX
- **Ideas** - Propón nuevas características y mejoras
- **Q&A** - Obtén ayuda con preguntas técnicas
- **Show and Tell** - Comparte tus proyectos usando AIOX
- **Troubleshooting** - Obtén ayuda con problemas
- **Squads** - Discusiones sobre AIOX Squads (equipos de agentes modulares)
- **MCP Ecosystem** - Discusiones sobre herramientas e integraciones MCP

[Únete a la Discusión](https://github.com/SynkraAI/aiox-core/discussions)

### Issue Tracker

Para reportes de bugs y solicitudes de características:

- [aiox-core Issues](https://github.com/SynkraAI/aiox-core/issues) - Framework principal
- [aiox-squads Issues](https://github.com/SynkraAI/aiox-squads/issues) - AIOX Squads
- [mcp-ecosystem Issues](https://github.com/SynkraAI/mcp-ecosystem/issues) - Herramientas MCP

## Cómo Contribuir

### Contribuciones de Código

1. Haz fork del repositorio
2. Crea una rama de característica (`git checkout -b feature/amazing-feature`)
3. Realiza tus cambios
4. Ejecuta las pruebas (`npm test`)
5. Envía un Pull Request

Consulta [CONTRIBUTING.md](CONTRIBUTING.md) para guías detalladas.

### Contribuciones sin Código

Valoramos todos los tipos de contribuciones:

- **Documentación** - Corrige errores tipográficos, mejora explicaciones
- **Traducción** - Ayuda a traducir la documentación
- **Reportes de Bugs** - Reporta problemas que encuentres
- **Ideas** - Comparte tus pensamientos sobre mejoras
- **Diseño** - Sugerencias de UI/UX
- **Promoción** - Posts de blog, charlas, tutoriales

### Squads

¡Crea y comparte tus propios Squads (equipos de agentes de IA modulares)!

Los Squads son equipos especializados de agentes de IA que trabajan juntos en dominios específicos:
- **ETL Squad** - Recolección y transformación de datos
- **Creator Squad** - Generación de contenido

Consulta [docs/Squads.md](docs/Squads.md) para detalles sobre cómo crear tu propio Squad.

## Gobernanza

### Toma de Decisiones

- **Decisiones menores**: Los mantenedores pueden decidir
- **Decisiones mayores**: Discutidas en GitHub Discussions
- **Cambios importantes**: Requieren proceso RFC

### Proceso RFC

Para cambios significativos:

1. Abre una Discussion con prefijo `[RFC]`
2. La comunidad proporciona retroalimentación
3. Los mantenedores toman la decisión final
4. La decisión es documentada

### Código de Conducta

Seguimos el [Contributor Covenant](CODE_OF_CONDUCT.md). Por favor léelo y respétalo.

Reporta violaciones a: conduct@SynkraAI.com

## Obteniendo Ayuda

1. Revisa la [Documentación](docs/)

## Hoja de Ruta del Proyecto

¿Quieres saber hacia dónde se dirige AIOX? Consulta nuestra hoja de ruta pública:

- [ROADMAP.md](ROADMAP.md) - Visión de alto nivel y características planificadas
- [GitHub Project](https://github.com/orgs/SynkraAI/projects/1) - Tablero de seguimiento detallado

La hoja de ruta se actualiza mensualmente y refleja la opinión de la comunidad. Puedes influir en nuestra dirección mediante:

1. **Votando** en ideas en [Discussions](https://github.com/SynkraAI/aiox-core/discussions/categories/ideas)
2. **Proponiendo** nuevas características a través del [proceso RFC](/.github/RFC_TEMPLATE.md)
3. **Contribuyendo** directamente a características planificadas

> Los elementos de la hoja de ruta son planes, no compromisos. Las prioridades pueden cambiar según las necesidades de la comunidad y las restricciones técnicas.

### Proceso de Solicitud de Características

Tenemos un proceso estructurado para proponer nuevas características:

1. **Ideas Rápidas** - Abre una Discussion en la categoría "Ideas"
2. **Proceso RFC** - Para características significativas, escribe un RFC usando nuestra [plantilla](/.github/RFC_TEMPLATE.md)
3. **Votación de la Comunidad** - Usa reacciones :+1: para mostrar apoyo
4. **Implementación** - Las ideas aprobadas pasan a nuestro backlog

Consulta [Proceso de Solicitud de Características](docs/FEATURE_PROCESS.md) para detalles completos.
