| Componente             | Descrição                                                             |
| ---------------------- | --------------------------------------------------------------------- |
| **Docker MCP Toolkit** | Servidores MCP pré-configurados executando em containers Docker       |
| **MCP Presets**        | Configurações curadas para diferentes workflows (dev, research, etc.) |
