### 此仓库中的内容

| 组件                 | 描述                                                          |
| ------------------------- | -------------------------------------------------------------------- |
| 🐳 **Docker MCP 工具包** | 在 Docker 容器中运行的预配置 MCP 服务器              |
| ⚙️ **MCP 预设**        | 为不同工作流策划的配置（开发、研究等）       |

| 🌐 **网关服务**    | 用于可靠 MCP 连接的 HTTP/SSE 网关                        |
